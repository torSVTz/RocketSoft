== Description ==

Tutor is a highly customized, responsive, elegant and retina ready education WordPress theme for school colleges and organizations. The best part of this theme is an elegant design which gives your institutional website a modern look. This theme has all the ability to control its options and make an amazing website for school, colleges or for small business. Its a perfect theme for sharing and selling your knowledge online. This theme is tested with the latest version of WordPress and some useful plugins like wocommerce, jetpack and contact form 7.
